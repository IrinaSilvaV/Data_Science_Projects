# Исследование данных о продажах игр
**Цель исследования** — выявить определяющие успешность игры закономерности, чтобы сделать ставку на потенциально популярный продукт и спланировать рекламные кампании.<br>
Перед нами данные за 2016 год.<br>
Описание данных на основании которых был проведен анализ: <br>
`Name` — название игры<br>
`Platform` — платформа<br>
`Year_of_Release` — год выпуска<br>
`Genre` — жанр игры<br>
`NA_sales` — продажи в Северной Америке<br>
`EU_sales`— продажи в Европе<br>
`JP_sales` — продажи в Японии<br>
`Other_sales` — продажи в других странах<br>
`Critic_Score` — оценка критиков<br>
`User_Score` — оценка пользователей<br>
`Rating` — рейтинг от организации ESRB.
